Welcome to the HP-API!

This is a work in progress, with much more data to be added.

For now, use these routes to request data on Harry Potter characters:

  • http://hp-api.herokuapp.com/characters - view all characters
  • http://hp-api.herokuapp.com/characters/students - view all characters who are Hogwarts students during the book series
  • http://hp-api.herokuapp.com/characters/house/:house - view all characters in a certain house, e.g. /slytherin
